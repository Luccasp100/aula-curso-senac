atividade sobre variáveis

declare variáveis para descrever um produto vendido no petshop. Você deve criar:

  1. uma constante para nome do produto (ex: "ração seca para cães adultos")
  2. uma variável let para o preço do produto (ex: 129.90)
  3. Uma variável let para a quantidade em estoque (ex: 35)
  4. uma constante para o código de barra do produto (ex:1234567890)
  5. uma variável let para indicar se o produto está em promoção (true ou false)
  6. mostre o tipo de dado de 3 valores diferentes
  7. crie uma frase completa usando concatenação que diga: "o produto [nome do produto] custa R$[preco] e temos [quantidade] em estoque
